# Tongram Messenger

Tongram is a messaging application built on Telegram's open-source codebase, enhanced with AI-powered features.

## Features

  ### Released (v1.0)
  - AI Translation — 30+ languages, one-tap in any chat
  - TonTV — Curated video content Mini App
  - Mini Apps — Organized directory of Telegram Mini Apps
  - Custom UI/UX — Distinct branding and color system

  ### In Development
  - AI Writing Assistant (grammar, tone adjustment)
  - Smart Folders (AI-powered chat organization)
  - Chat Summary (AI conversation summaries)
